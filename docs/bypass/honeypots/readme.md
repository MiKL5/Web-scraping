# **La estion des Honeypots**<a href="../../../"><img align="right" src="../../../assets/atomicWebScraping.png" alt="Web scraping" height="64px"></a>
Les honeypots sont des liens ou des éléments cachés dans le code HTML qui ne sont pas visibles pour les utilisateurs humains, mais que les bots peuvent suivre [1] [2]. Si un bot accède à un honeypot, il est immédiatement bloqué [2].

Inspecter le code source HTML pour s'assurer que le scraper n'interagit pas avec des éléments cachés ou des liens suspects [2]. Ne pas cliquer sur les liens utilisant des styles CSS tels que display: none ou visibility: hidden [2].
