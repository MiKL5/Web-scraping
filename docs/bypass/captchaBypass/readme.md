# **Le contournement des CAPTCHA**<a href="../../../"><img align="right" src="../../../assets/atomicWebScraping.png" alt="Web scraping" height="64px"></a>
Les CAPTCHA (Completely Automated Public Turing test to tell Computers and Humans Apart) sont conçus pour distinguer les humains des bots [2][3].

Utiliser des services de résolution de CAPTCHA qui emploient des humains pour résoudre les CAPTCHA à la demande [2].  
Intégrer des solveurs de CAPTCHA avec des navigateurs sans tête : Utiliser des outils comme Selenium ou Puppeteer pour automatiser la résolution des CAPTCHA [2].
