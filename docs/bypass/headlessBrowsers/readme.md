# **Les Headless Browsers**<a href="../../../"><img align="right" src="../../../assets/atomicWebScraping.png" alt="Web scraping" height="64px"></a>
Les sites web dynamiques utilisent JavaScript pour charger du contenu, ce qui peut rendre difficile l'extraction des données avec des requêtes HTTP simples [2].

Utiliser des navigateurs sans tête pour rendre la page web complète, y compris le contenu généré par JavaScript [2]. Ces outils peuvent exécuter JavaScript et simuler les interactions d'un utilisateur réel [2].
