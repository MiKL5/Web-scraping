# **Analyser le comportement de l'utilisateur (UBA)**<a href="../../../"><img align="right" src="../../../assets/atomicWebScraping.png" alt="Web scraping" height="64px"></a>
Les sites web peuvent analyser le comportement de l'utilisateur pour détecter des anomalies qui suggèrent une activité de bot [1][2].

Simuler un comportement humain en effectuant des actions telles que le déplacement de la souris, le défilement de la page et les clics aléatoires [2].  
Éviter les clics statiques ou les actions répétitives qui peuvent signaler une automatisation [2].
