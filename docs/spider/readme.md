# **Qu'est-ce qu'un Spider**<a href="../../"><img align="right" src="../../assets/atomicWebScraping.png" alt="Web scraping" height="64px"></a>
<h3>Un spider dans le contexte du web scraping est un programme automatisé, aussi nommé robot ou bot. Son principal but est de parcourir systématiquement le web. Il navigue depuis l’URL initiale, suit les liens (le crawling) et collecte le contenu pour en extraire les données (le parsing).</h3>  

En web scraping, les spiders sont indispensables pour automatiser la collecte massive de données à partir de sites web en suivant des règles prédéfinies.
Le processus typique comporte trois étapes principales :  
1. Le fetching (la requête). Le spider télécharge les pages web ciblées.  
2. Le crawling. Explorer systématique des pages du site par suivi des liens.  
3. Le parsing. Analysé le contenu, piis extraire les infos voulues, souvent à l’aide de sélecteurs CSS ou XPath.  

Les moteurs de recherche les utilisent pour indexer le web, mais dans le cadre du web scraping, ils servent à collecter des données utiles pour des analyses, surveillances ou projets automatisés. Les données récoltées par les spiders sont enregistrées sous des formats exploitables comme JSON ou CSV.  
Généralement, ils respectent les règles spécifiées par les fichiers robots.txt des sites pour éviter le scraping non autorisé. Ils combinent rapidité de collecte et contrôle précis sur les informations extraites, ce qui en fait un outil clé du web scraping moderne.
___
Le web scraping illégal expose à d'importants risques juridiques.
* En droit pénal français, l’article 323-3 du Code pénal _punit de 5 ans d’emprisonnement et de 150 000 euros d’amende le fait d’extraire frauduleusement des données ou d’accéder illicitement à un système automatisé_. L’intention frauduleuse doit être prouvée.
* En droit de la propriété intellectuelle, _le scraping de contenu protégé sans autorisation peut entraîner des sanctions allant jusqu’à 5 ans de prison et 300 000 euros d’amende_.
* Le _non-respect du RGPD lors de la collecte de données personnelles par scraping peut conduire à des amendes, allant jusqu’à 20 millions d’euros ou 4% du chiffre d’affaires mondial de l’entreprise_, en plus d'autres sanctions.
* Le scraping peut aussi constituer une violation des conditions générales d’utilisation (CGU) d’un site web, ouvrant la voie à des poursuites civiles pour rupture de contrat et potentiellement des actions en concurrence déloyale.
* D'autres conséquences peuvent inclure le blocage d’adresses IP, des mises en demeure, et des actions judiciaires selon les juridictions visées.
